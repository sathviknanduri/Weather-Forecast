# SkyCast: High-Precision Weather Dashboard 🌤️

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com/)

A modern, responsive weather application providing real-time meteorological data with a focus on clean UI and performant data fetching. Built using the **Vite + React** ecosystem for lightning-fast load times and type safety via **TypeScript**.

---

## 🚀 Key Features

* **Real-Time Data Fetching:** Integrated with global weather APIs to provide up-to-the-minute temperature, humidity, and wind speed.
* **Location Intelligence:** Automatic geolocation detection and a robust search feature for worldwide city coverage.
* **Dynamic Visuals:** UI components that adapt based on weather conditions (Clear, Rain, Snow, etc.) using Tailwind CSS.
* **Extended Forecast:** Detailed 5-day/7-day forecasts visualized through sleek, interactive cards.
* **Premium UI Components:** Built with **shadcn/ui** for accessible, high-quality interface elements like modals, tooltips, and cards.

---

| Layer | Technology |
